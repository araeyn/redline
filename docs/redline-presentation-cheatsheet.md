# Redline Presentation Cheatsheet

## 20-second opener

Redline is an AI art director for works in progress. Instead of generating art for creators, it mentors them while they create by critiquing composition, simulating lighting, checking readability, and identifying exactly what to fix next.

## Best one-line pitch

Redline helps artists make stronger decisions in the messy middle of the creative process.

## If a judge asks what is innovative

Most AI art projects automate creation. Redline augments artistic judgment. That is a different and less common direction: AI as mentor, not replacement.

## If a judge asks why people would use it

Artists lose the most time when they render too early. Redline catches weak composition, unclear focus, and lighting issues before that time gets wasted.

## If a judge asks about the target user

Digital artists, illustrators, concept artists, and students working on sketches, paintings, environment art, props, comics, or stylized pieces.

## If a judge asks about the strongest feature

The strongest feature is the loop:

1. Analyze the full piece.
2. Generate a next-pass plan.
3. Box one weak region.
4. Run a local mentor pass just for that area.

That feels like a real workflow, not a gimmick.

## Best 60-second demo script

Start with a WIP already loaded.  
Run Full Mentor Pass.  
Point to the critique markers and the Next Pass Plan.  
Switch to Relight and drag the light source.  
Open Readability and show the silhouette and thumbnail ladder.  
Use Focus Region on one weak area and run Analyze Region.  
Close with: Redline turns vague artistic uncertainty into concrete next actions.

## Final closing line

We did not build AI to replace artists. We built AI to help artists finish better work.
